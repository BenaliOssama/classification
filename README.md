# Classification

Practical classification with Scikit-learn — logistic regression, decision boundaries, and model evaluation.

## Setup

```bash
python -m venv ex00
source ex00/bin/activate
pip install -r ex00/requirements.txt
```

## Exercises

| Exercise | Topic |
|----------|-------|
| ex00 | Environment setup |
| ex01 | Logistic regression basics |
| ex02 | Sigmoid function |
| ex03 | Decision boundary |
| ex04 | Train/test split and stratification |
| ex05 | Breast cancer prediction |

## Run

```bash
python ex01/ex01.py
python ex02/ex02.py
python ex03/ex03.py
python ex04/ex04.py
python ex05/ex05.py
```

## ex05 Data

Place the dataset files in `ex05/data/`:
- `breast-cancer-wisconsin.data`
- `breast-cancer-wisconsin.names`
